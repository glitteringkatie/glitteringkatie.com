## Preview

To run locally:

- `yarn`
- `yarn dev`
- Open [https://localhost:3000](https://localhost:3000)

To use your own google analytics:

- `cp .example-env .env`
- Replace `YOUR_GOOGLE_ANALYTICS_TRACKING_ID` with your tracking ID
